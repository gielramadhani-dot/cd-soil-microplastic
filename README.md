# Microplastic-Controlled Cadmium Mobilization in Agricultural Soils

This repository contains the complete computational workflow, processed datasets, figures, and analysis outputs for a mechanism-informed study of microplastic-controlled cadmium mobilization in agricultural soils.

The workflow integrates hydrated Cd–microplastic molecular descriptors, Microplastic Reactivity Index, Soil Geochemical Modifier, Cadmium Mobilization Index, Monte Carlo uncertainty analysis, sensitivity analysis, network pathway analysis, and volcano-style risk-enrichment visualization.

CMI is used as a relative scenario-screening index and should not be interpreted as a measured Cd concentration.

STAGE 10B-PUBLICATION VOLCANO-STYLE CMI RISK-ENRICHMENT REPORT
================================================================================

Purpose:
This analysis creates a volcano-style landscape for deterministic CMI scenario data.

Point definition:
Each point is an individual sampled Cd mobilization scenario.

x-axis:
log2 fold change of CMI relative to matched median CMI.
Matching variables: pH, organic matter, particle size, and Cd total.

y-axis:
Empirical tail-strength score calculated from directional tail probability within matched scenario groups.

Thresholds:
log2FC threshold = ±0.6
Tail-strength threshold = 1.3
Y-axis cap = 6.0

Class counts:
- not strongly differentiated: 252595
- enriched mobilization risk: 99643
- reduced mobilization risk: 97762

Interpretation caution:
This is not a statistical gene-expression volcano plot.
It is a volcano-style risk-enrichment visualization for scenario-based CMI modeling.
The y-axis should be described as an empirical tail-strength score, not as a p-value.

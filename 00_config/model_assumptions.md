# Model assumptions

1. Microplastic fragments are represented using small molecular fragments.
2. Fresh PE is represented as a decane-like alkane oligomer.
3. Oxidized PE is represented as a carboxylated alkane fragment.
4. Fresh PET is represented as an aromatic diester-containing fragment.
5. Oxidized PET is represented as an ester/carboxyl-containing aromatic fragment.
6. Cadmium is represented as [Cd(H2O)6]2+ to approximate hydrated Cd in soil water.
7. xTB screening is used to rank initial Cd–microplastic binding poses.
8. DFT refinement will be conducted later only on the lowest-energy chemically reasonable xTB structures.
9. xTB/DFT descriptors are used as molecular descriptors, not direct field-scale predictions.
